Uganda Knuckles Gun:

Description:

Basing on Memes, this weapon you can shoot rings against zombies using a uganda knuckles gun;

Cvars:

Cvars: 
zp_uganda_dmg "90.0" // Weapon Damage
zp_uganda_recoil "0.3" // Weapon Recoil
zp_uganda_clip "30" // Weapon Clip
zp_uganda_ammo "180" // Weapon Ammo
zp_uganda_ringlife "2.0" // Time for remove ring entity
zp_uganda_ringspeed "2100" // Speed of ring
