[ZPSp] Addon: Class Limiter

Description:

This addon allows limiting the number of zombie/human classes used by players at the same time. This plugin automatically adds to classes' .ini files a "LIMIT" field that you can change to set limits for each class separately.

Requeriments:

  • Amx 1.9 or higher
  • Zombie Plague Special 4.5 or higher

How to use:

When plugin enables, will be add a limit line in zpsp_zombieclasses.ini or zpsp_humanclasses.ini and you can change limit for what you want.
