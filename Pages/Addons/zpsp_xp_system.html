[ZPSp] Addon: Xp System

Description:

This plugin turns your zombie plague special 4.5 into zombie plague XP (With Level and Upgrade) without edit main plugin.

Requeriments:

Amxmodx 1.9 or higher

Zombie Plague Special 4.5 [Patch 1]

How to use require level for a class/item/weapon:

When plugin enables, will be add a level line in a zombie_plague_special.ini, zpsp_zombieclasses.ini, zpsp_humanclasses.ini, zpsp_extraitems.ini and zpsp_custom_weapons.ini and you can change minimum level require for use this item/class/weapon

Cvars:

  • zp_xp_save_type "1" ; Save Data Type: (1 - Authid | 2 = Name | 3 = IP)
  • zp_xp_for_kill "20" ; Xp For Kill Zombies/Humans
  • zp_xp_for_kill_specials "30" ; Xp For Kill Zombies/Humans Specials
  • zp_xp_infect "20" ; Xp For Infect Players
  • zp_xp_damage_give "2" ; Xp for X damage
  • zp_xp_damage_needed "750" ; Needed damage for give xp by damage
  • zp_xp_vip_flag "b" ; Flag Acess to Bonus XP
  • zp_xp_vip_bonus_xp "10" ; Bonus XP Amount
