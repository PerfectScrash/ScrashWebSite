[ZPSp] Addon: Custom Player Model

Description:

Give for any user a Custom Player Model/Skin by Nick/Ip/Steam ID/Flag Acess

Requeriments:

Latest Version of Zombie Plague Special

Adding Player Model:

For adding a Player Model first at all open your zpsp_player_skins.ini and add this

If player model are by Flag:

"flag" "y" "owner_human" "owner_zombie"
Note: With this you added a model human and zombie for players that have flag "y";

If player model are by Nick:

"nick" "[P]erfec[T] [S]cr[@]s[H]" "perfect_ct_naruto" "!default"
Note: With this you added only one human model for nick [P]erfec[T] [S]cr[@]s[H];

If player model are by Steam ID:

"steam" "STEAM_0:0:00000000" "steam_nice" "zombie_steam"
Note: With this you added a model human and zombie for player that have steam id: "STEAM_0:0:00000000";

If player model are by IP:

"ip" "177.93.106.8" "!default" "skin_zm_fat"
Note: With this you added only one zombie model for players that have ip: 177.93.106.8;
